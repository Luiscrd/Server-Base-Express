# Server-Base-Express
Servidor base creado en expres para podert ser reutilizado según necesidades.

## Instrucciones

Reconstuir módulos de Node
```
npm i
```

Generar el DIST
```
tsc -w
```

Levantar servidor
```
nodemon dist/
```
## Librerias

* TypeScript
* Node
* Nodemon
* Express
* Cors
